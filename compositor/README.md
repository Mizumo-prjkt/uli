# Compositor for Universal Linux Installer

YAML compositor for Universal Linux Installer

License: LGPL v2


This is a YAML compositor for Universal Linux Installer. It is a C++ program that uses the YAML-CPP library to parse YAML files and compose them into a single YAML file. It is used to compose the YAML files for the Universal Linux Installer.

It also fetches distro's repository from a remote server and composes it into a single YAML file. (As a translation and/or file that can be used for other distros)

This component is OS agnostic, at least utilizing Qt Technology. Should work both Windows and Linux
