A tool to allow unified installation of Linux distributions.

---

As Linux is slowly getting popular, due to complaints that Windows is getting very sloppy at their work, many users flock to Linux distributions. However, there are many distributions out there, and it can be difficult to install every distribution for those who wanted to try out flavors, because of different packaging naming schemes, and whatnot.

This tool aims to unify the naming convention at surface, and let the program does the rest, it can also serve as a tool to easily install a linux distribution in an unattended way.

## Components Used

- C/C++
- Qt for UI
- CMake
